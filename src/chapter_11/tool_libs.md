##  ouch： 压缩和解压缩工具

`ouch`代表**Obvious Unified Compression Helper**，是一个 CLI 工具，可帮助您压缩和解压缩多种格式的文件。

| 格式   | `.tar` | `.zip` | `.bz`, `.bz2` | `.gz` | `.lz4` | `.xz`, `.lzma` | `.zst` |
| ------ | ------ | ------ | ------------- | ----- | ------ | -------------- | ------ |
| 支持的 | ✓      | ✓      | ✓             | ✓     | ✓      | ✓              | ✓      |

[https://github.com/ouch-org/ouch](https://github.com/ouch-org/ouch)
